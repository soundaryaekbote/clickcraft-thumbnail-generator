# 🎨 ClickCraft - AI Thumbnail Generator

ClickCraft is a full-stack AI-powered web application that helps users create eye-catching thumbnails instantly using modern web technologies.

---

## 🚀 Features

* 🎯 Generate high-quality thumbnails using AI
* ⚡ Fast and responsive UI
* 🧠 Full-stack architecture (Frontend + Backend)
* 🔐 Secure environment variables handling
* 📦 Scalable project structure

---

## 🛠 Tech Stack

### Frontend

* React / Vite
* HTML, CSS, JavaScript

### Backend

* Node.js
* Express.js

---

## 📁 Project Structure

```
thumblify_fullstack/
│── client/      # Frontend (React)
│── server/      # Backend (Node.js)
│── .gitignore
│── README.md
```

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/clickcraft-thumbnail-generator.git
```

### 2. Install dependencies

#### Client

```
cd client
npm install
```

#### Server

```
cd ../server
npm install
```

---

## ▶️ Run the project

### Start backend

```
cd server
npm start
```

### Start frontend

```
cd client
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in the server folder and add:

```
YOUR_API_KEY=your_api_key_here
```

⚠️ Do NOT share your `.env` file publicly.
